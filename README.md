# Curriculum Vitae Interactivo - Maximiliano Janzen

Un curriculum vitae moderno e interactivo con componentes animados, integrado con componentes de diseño de **21st.dev/community/components**.

## 📋 Características

✨ **Diseño Moderno**
- Interfaz oscura y elegante con gradientes
- Componentes animados y responsivos
- Tailwind CSS para estilos modernos

🎨 **Componentes Interactivos**
- **Background Boxes**: Fondo animado con cajas que cambian de color al pasar el mouse
- **Marquee de Testimonios**: Scroll continuo de recomendaciones profesionales
- **AI Chat Bot**: Sección de preguntas frecuentes con chat interactivo

📱 **Totalmente Responsivo**
- Diseño mobile-first
- Optimizado para todos los dispositivos

## 🚀 Cómo Usar

### Opción 1: HTML Puro (Recomendado para vista rápida)

```bash
# Simplemente abre el archivo index.html en tu navegador
open index.html
```

El archivo `index.html` es una versión completamente funcional y autónoma que no requiere instalación de dependencias.

### Opción 2: React + Vite (Desarrollo)

```bash
# Instalar dependencias
npm install

# Desarrollo
npm run dev

# Build para producción
npm run build
```

## 📁 Estructura del Proyecto

```
.
├── index.html                 # Versión HTML pura (recomendada)
├── src/
│   ├── App.jsx               # Componente principal React
│   ├── main.jsx              # Entrada de React
│   ├── index.css             # Estilos globales
│   └── components/
│       └── ui/
│           ├── background-boxes.jsx  # Componente de fondo animado
│           ├── marquee.jsx          # Componente marquee para scroll
│           └── ai-chat.jsx          # Componente chat IA
├── react-index.html          # HTML para React + Vite
├── tailwind.config.js        # Configuración Tailwind CSS
├── vite.config.js            # Configuración Vite
└── postcss.config.js         # Configuración PostCSS

```

## 🎯 Contenido del Curriculum

### Secciones Incluidas:
1. **Perfil Profesional** - Resumen de experiencia y especialidades
2. **Experiencia Laboral** - Historial detallado de empleos
3. **Habilidades Técnicas** - Competencias en DevOps, Cloud, Ciberseguridad
4. **Testimonios Animados** - Recomendaciones con marquee vertical
5. **Chat Interactivo** - Sección de preguntas y respuestas
6. **Educación** - Formación académica
7. **Certificaciones** - Cursos y certificados profesionales

## 🎨 Personalización

### Cambiar Colores
Edita `tailwind.config.js` para modificar la paleta de colores principal.

### Agregar Nuevas Secciones
Modifica `src/App.jsx` (React) o `index.html` (HTML puro) para agregar más contenido.

### Ajustar Animaciones
- Para Boxes: `src/components/ui/background-boxes.jsx`
- Para Marquee: `src/components/ui/marquee.jsx`
